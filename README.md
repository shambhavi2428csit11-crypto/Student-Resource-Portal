# Student Resource Portal

A student-friendly web platform built using **HTML**, **CSS**, **JavaScript**, and **Node.js**.
This project helps students access notes, coding resources, cloud learning links, placement preparation material, and important academic resources in one place.

## 🚀 Project Overview

The **Student Resource Portal** is designed to make academic and learning resources easily accessible for students.
Instead of searching different websites for notes, coding practice, cloud resources, and placement preparation, students can use this platform to find everything in an organized way.

The project includes responsive UI pages and backend routes to manage and display resource data category-wise.

## ✨ Features

* Access academic notes in one place
* Explore coding resources and practice links
* View cloud learning resources
* Find placement preparation material
* Category-wise organization of resources
* Responsive and user-friendly interface
* Backend routes for managing resource data
* Simple and beginner-friendly design

## ⚙️ Installation and Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Student-Resource-Portal.git
```

### 2. Go to the project folder

```bash
cd Student-Resource-Portal
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the project

```bash
node server.js
```

### 5. Open in browser

```bash
http://localhost:3000
```

## 🧠 How It Works

1. User opens the student resource portal.
2. Resources are displayed in different categories.
3. Students can choose a category such as subjects, coding, cloud, or placement preparation.
4. The backend handles routes for sending resource data.
5. The frontend displays the resources in a clean and organized layout.
